Namespace JsonApiDotNetCore.Diagnostics

Classes

AspNetCodeTimerSession

Code timing session management intended for use in ASP.NET Web Applications. Uses Microsoft.AspNetCore.Http.HttpContext.Items to isolate concurrent requests. Can be used with async/wait, but it cannot distinguish between concurrently running threads within a single HTTP request, so you'll need to pass an JsonApiDotNetCore.Diagnostics.CascadingCodeTimer instance through the entire call chain in that case.

CodeTimingSessionManager

Provides access to the "current" measurement, which removes the need to pass along a JsonApiDotNetCore.Diagnostics.CascadingCodeTimer instance through the entire call chain.

DefaultCodeTimerSession

General code timing session management. Can be used with async/wait, but it cannot distinguish between concurrently running threads, so you'll need to pass an JsonApiDotNetCore.Diagnostics.CascadingCodeTimer instance through the entire call chain in that case.

Interfaces

ICodeTimer

Records execution times for code blocks.

ICodeTimerSession

Removes the need to pass along a JsonApiDotNetCore.Diagnostics.CascadingCodeTimer instance through the entire call chain when using code timing.
